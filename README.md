# Java Web Page Inspector (Инспектор Веб-Страниц Java)

Java Web Page Inspector - это простой инструмент командной строки, который помогает вам интерактивно исследовать веб-страницы. Он предоставляет информацию о видимых и кликабельных элементах на веб-странице и позволяет выполнять действия, такие как клик по элементам. Для автоматизации и взаимодействия с веб-страницами этот инструмент использует Playwright.

## Особенности

- Отображение видимых элементов на веб-странице
- Отображение кликабельных элементов на веб-странице
- Клик по любому кликабельному элементу и навигация по веб-странице
- Настройка поведения инструмента с помощью параметров командной строки

## Классы

### Основной класс: PlaywrightExample

Этот класс содержит основной метод и организует взаимодействие с веб-страницей с использованием следующих классов:

- `Options`: Хранит пользовательские настройки и предпочтения для утилиты
- `OptionsService`: Обрабатывает разбор и установку параметров командной строки для утилиты
- `PageScanner`: Сканирует веб-страницу на предмет видимых и кликабельных элементов
- `ElementAction`: Обрабатывает действия, такие как клик по элементам
- `ElementPrinter`: Выводит на консоль видимые и кликабельные элементы
- `UserInteraction`: Обрабатывает пользовательский ввод для взаимодействия с утилитой

### Options

Класс `Options` хранит предпочтения пользователя для утилиты. Он имеет следующие атрибуты:

- `showVisible`: Если установлено значение `true`, утилита будет отображать видимые элементы на веб-странице
- `showClickable`: Если установлено значение `true`, утилита будет отображать кликабельные элементы на веб-странице
- `askForClick`: Если установлено значение `true`, утилита будет предлагать пользователю кликнуть по элементу
- `navigateLink`: URL-адрес веб-страницы для перехода

### OptionsService

Класс `OptionsService` отвечает за разбор параметров командной строки и установку соответствующих атрибутов в классе `Options`. Он поддерживает следующие параметры командной строки:

- `-l`: Устанавливает URL для перехода (например, `-l=https://example.com`)
- `-v`: Переключает отображение видимых элементов на веб-странице
- `-c`: Переключает отображение кликабельных элементов на веб-странице
- `-a`: Переключает возможность предложения пользователю кликнуть по элементу

### PageScanner

Класс `PageScanner` отвечает за сканирование веб-страницы на предмет видимых и кликабельных элементов. Он предоставляет следующие методы:

- `scanPageForVisible()`: Возвращает список видимых элементов на веб-странице
- `scanPageForClickable()`: Заполняет список кликабельных элементов на веб-странице

### ElementAction

Класс `ElementAction` отвечает за выполнение действий с элементами веб-страницы, таких как клик по элементу. Он предоставляет метод `clickOnElement()`, который принимает список кликабельных элементов, индекс элемента для клика и текст для ввода в случае, если элемент является полем ввода.

### ElementPrinter

Класс `ElementPrinter` выводит на консоль информацию о видимых и кликабельных элементах. Он предоставляет следующие методы:

- `printVisible()`: Выводит список видимых элементов на веб-странице
- `printClickable()`: Выводит список кликабельных элементов на веб-странице с указанием их типов

### UserInteraction

Класс `UserInteraction` отвечает за обработку пользовательского ввода для взаимодействия с утилитой. Он предоставляет метод `askForClick()`, который принимает список типов кликабельных элементов и запрашивает у пользователя индекс элемента для клика и текст для ввода (если элемент является полем ввода).

## Использование

Для использования утилиты скомпилируйте и запустите файл `PlaywrightExample.java` с соответствующими параметрами командной строки.

Пример:

```
java PlaywrightExample -l=https://example.com -v -c -a
```

Эта команда запустит утилиту с указанными параметрами:

- URL для перехода: `https://example.com`
- Отображение видимых элементов: включено
- Отображение кликабельных элементов: включено
- Предложение пользователю кликнуть по элементу: включено

Утилита будет выводить информацию о видимых и кликабельных элементах на веб-странице и предложит пользователю кликнуть по одному из них.
